## Installation

### Prerequisites:

1. **Node.js:** [Download and Install Node.js](https://nodejs.org/)

### Getting Started:

```bash
# Clone the repository
git clone https://github.com/Alaeddine-Nasri/UsineOccupation.git

# Navigate to the project directory
cd Occupancy-Usine

# Install dependencies
npm install

#Run the project
npm run
```
